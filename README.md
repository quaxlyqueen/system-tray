# A System Tray for i3 and i3blocks
A system tray written in Python for the GUI.

# Installation:
`git clone https://github.com/quaxlyqueen/system-tray`<br>
`cd system-tray`<br>
`sudo ./install`<br>

# Uninstall:
`sudo ./uninstall`<br>
